### Common Props

| Name     | Type    | Default | Description                                            |
| -------- | ------- | ------- | ------------------------------------------------------ |
| `padded` | boolean | `false` | 在分隔线的每一侧都留有间隙，以匹配最后一个图块的填充。 |
| `inset`  | boolean | `false` | 增加分隔线的前导边距，以使其不与图标或图像列相交。     |
