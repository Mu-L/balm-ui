```html
<ui-card-actions><!-- card actions area --></ui-card-actions>
```

### Props

| Name        | Type    | Default | Description                                    |
| ----------- | ------- | ------- | ---------------------------------------------- |
| `fullBleed` | boolean | `false` | 删除操作区域的填充，并使其操作区域宽度为 100％ |

### Slots

| Name      | Props | Description                  |
| --------- | ----- | ---------------------------- |
| `default` |       | default 插槽包含卡片动作组件 |

**子组件：**

- `<ui-card-buttons>`
- `<ui-card-icons>`
