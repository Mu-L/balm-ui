```html
<ui-card><!-- Card contents --></ui-card>
```

### Props

| Name       | Type    | Default | Description            |
| ---------- | ------- | ------- | ---------------------- |
| `outlined` | boolean | `false` | 去除阴影并显示细线轮廓 |

### Slots

| Name      | Props | Description                       |
| --------- | ----- | --------------------------------- |
| `default` |       | default 插槽包含卡片子组件及 HTML |

**子组件：**

- `<ui-card-content>`
- `<ui-card-actions>`
