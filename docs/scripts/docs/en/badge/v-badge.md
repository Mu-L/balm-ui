## Badge directive

```html
<div v-badge></div>
```

- `v-badge="count"`
- `v-badge.overlap="count"`
- `v-badge.dot`

| Value   | Type   | Default | Description              |
| ------- | ------ | ------- | ------------------------ |
| `count` | number | `0`     | Number to show in badge. |

> maximum count: `99`
